About me

愛媛県出身。
高校卒業後、地元の病院に医療事務として就職。
同じ毎日の繰り返しに嫌気が差し、様々なことに挑戦する人生にしようと決意。
様々な投資やビジネスに手を出すも上手くいかず。
高校時代、野球していたこともあり、地道な努力を繰り返して結果を出した経験から
地道な努力と確かなスキルで勝負するしかないと思い、プログラミングに興味を持つ。
その後ITの技術に惹かれ、プログラミングスクールTECH I.S.に入校。
フロントエンド・バックエンドの開発に必要なスキルやサーバー構築に関するスキルを学習し
要件定義からデバッグまですべて通して行うチーム開発に携わる。

スキル：HTML | CSS | Bootstrap | JavaScript | jQuery | Node.js | PHP | Laravel | MySQL
